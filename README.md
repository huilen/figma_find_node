# Requirements
- Python 3
- Virtualenv
# Install virtualenv
Go to the project directory and install the virtualenv this way:
```
virtualenv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
# Run project
From the project directory just run:
```
python main.py
```
# Run tests
From the project directory just run:
```
python -m unittest tests.test_main
```
